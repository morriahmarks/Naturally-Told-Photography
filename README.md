# Naturally Told Photography Website

This is a 5-page website created for a Communicating Through Web Design course.

## About the Project
This website is designed as a beginner-level photography portfolio for a fictional brand called "Naturally Told Photography." The focus is on candid photography that captures real-life moments without posed direction.

## Pages Included
- Home
- About
- Portfolio
- Blog
- Contact

## Technologies Used
- HTML
- CSS
- GitHub Pages (for hosting)

## Purpose
The goal of this project is to practice web design principles such as layout, navigation, visual hierarchy, and responsive design while learning how to communicate content effectively through a website.
